    PJM Cleaned dbt Views¶

    All views are materialized in the pjm_cleaned schema. The dbt pipeline follows a layered architecture:

    Source (raw tables) -> Staging (clean/rename) -> Marts (join/aggregate, final views)

    Source and staging models are ephemeral (not materialized as tables/views). Only mart models are exposed as views.

    Mart Views¶

    pjm_lmps_hourly¶

    Field Value
    Business Definition Hourly electricity prices (LMPs) combining day-ahead and real-time markets
    Grain One row per hour per pricing node
    Primary Keys datetime_beginning_utc, pnode_id
    Upstream pjm.da_hrl_lmps, pjm.rt_unverified_hourly_lmps, pjm.rt_settlements_verified_hourly_lmps
    Use Cases DA vs RT price spread analysis, node-level price tracking, congestion analysis
    Refresh Refreshes on query (view)

    pjm_lmps_daily¶

    Field Value
    Business Definition Daily average electricity prices by pricing node
    Grain One row per day per pricing node
    Primary Keys date, pnode_id
    Upstream staging_v1_pjm_lmps_daily (from hourly LMP sources)
    Use Cases Daily price trend analysis, settlement reconciliation

    pjm_lmps_rt_hourly¶

    Field Value
    Business Definition Real-time only hourly LMPs, combining verified and unverified
    Grain One row per hour per pricing node
    Primary Keys datetime_beginning_utc, pnode_id
    Upstream pjm.rt_unverified_hourly_lmps, pjm.rt_settlements_verified_hourly_lmps
    Logic Uses verified prices when available; falls back to unverified for recent data
    Use Cases Real-time price monitoring, intraday P&L

    pjm_load_da_hourly¶

    Field Value
    Business Definition Day-ahead cleared load by region (from demand bids)
    Grain One row per hour per region
    Primary Keys datetime_beginning_utc, region
    Upstream pjm.hrl_dmd_bids
    Use Cases Expected next-day demand, DA load vs actual comparison

    pjm_load_da_daily¶

    Field Value
    Business Definition Daily total day-ahead cleared load by region
    Grain One row per day per region
    Upstream staging_v1_pjm_load_da_daily

    pjm_load_rt_metered_hourly¶

    Field Value
    Business Definition Actual metered load by region
    Grain One row per hour per region
    Upstream pjm.hourly_load_metered
    Use Cases Ground truth load for forecast accuracy, settlement

    pjm_load_rt_metered_daily¶

    Field Value
    Business Definition Daily total metered load by region
    Grain One row per day per region

    pjm_load_rt_prelim_hourly¶

    Field Value
    Business Definition Preliminary real-time load by region (available same-day)
    Grain One row per hour per region
    Upstream pjm.hourly_load_prelim
    Use Cases Intraday load tracking before metered data is available

    pjm_load_rt_prelim_daily¶

    Field Value
    Business Definition Daily total preliminary load by region
    Grain One row per day per region

    pjm_load_rt_instantaneous_hourly¶

    Field Value
    Business Definition Hourly average of 5-minute instantaneous load
    Grain One row per hour per region
    Upstream pjm.five_min_instantaneous_load_v1_2025_oct_15
    Logic Aggregates 5-min readings to hourly averages

    pjm_load_forecast_hourly¶

    Field Value
    Business Definition PJM's official 7-day hourly load forecast by area
    Grain One row per hour per forecast area
    Upstream pjm.seven_day_load_forecast_v1_2025_08_13
    Use Cases Compare against Meteologica/trader forecasts, anticipate demand

    pjm_load_forecast_daily¶

    Field Value
    Business Definition Daily total from PJM's load forecast
    Grain One row per day per forecast area

    pjm_gridstatus_load_forecast_hourly¶

    Field Value
    Business Definition Load forecast sourced from GridStatus (alternative to PJM direct)
    Grain One row per hour per region
    Upstream gridstatus.pjm_load_forecast

    pjm_fuel_mix_hourly¶

    Field Value
    Business Definition Hourly electricity generation by fuel type
    Grain One row per hour per fuel type
    Upstream gridstatus.pjm_fuel_mix_hourly
    Use Cases Monitor gas vs coal switching, renewable penetration

    pjm_fuel_mix_daily¶

    Field Value
    Business Definition Daily total generation by fuel type
    Grain One row per day per fuel type

    pjm_outages_actual_daily¶

    Field Value
    Business Definition Actual daily generation outages by type (planned, maintenance, forced)
    Grain One row per day per region
    Upstream pjm.seven_day_outage_forecast
    Use Cases Track supply availability, compare actual vs forecast outages

    pjm_outages_forecast_daily¶

    Field Value
    Business Definition Forecasted daily generation outages
    Grain One row per day per region

    pjm_tie_flows_hourly¶

    Field Value
    Business Definition Hourly tie flows between PJM and neighboring regions
    Grain One row per hour per interface
    Upstream pjm.five_min_tie_flows
    Use Cases Import/export tracking, cross-ISO flow analysis

    pjm_tie_flows_daily¶

    Field Value
    Business Definition Daily net tie flows by interface
    Grain One row per day per interface

    pjm_solar_forecast_hourly¶

    Field Value
    Business Definition PJM solar generation forecast (front-of-meter and behind-the-meter)
    Grain One row per hour
    Upstream gridstatus.pjm_solar_forecast_hourly

    pjm_wind_forecast_hourly¶

    Field Value
    Business Definition PJM wind generation forecast
    Grain One row per hour
    Upstream gridstatus.pjm_wind_forecast_hourly

    Query Views (Analysis Helpers)¶

    These are intermediate analysis views in the pjm_cleaned schema:

    View Purpose
    query_v1_pjm_load_actual_vs_forecast Compares actual load against PJM's forecast for accuracy tracking
    query_v1_pjm_load_forecast_performance_daily Daily forecast error metrics
    query_v1_pjm_load_forecast_performance_hourly Hourly forecast error metrics

    Utility Models¶

    Model Purpose
    utils_v1_pjm_load_regions Maps load areas to PJM regions (MIDATL, WEST, SOUTH)
    utils_v1_pjm_dates_daily Date spine for daily models
    utils_v1_pjm_dates_hourly Date spine for hourly models
