    Glossary¶

    Business Terms¶

    Term Definition
    ISO Independent System Operator -- the entity that manages the electric grid in a region. Major U.S. ISOs: PJM, ERCOT, MISO, CAISO, NYISO, ISO-NE, SPP.
    LMP Locational Marginal Price -- the cost of electricity at a specific point on the grid, in $/MWh.
    DA (Day-Ahead) The market where electricity is bought and sold for delivery the next day. Prices are set by ~1 PM the day before.
    RT (Real-Time) The market where electricity is bought and sold for immediate delivery. Prices are set every 5 minutes.
    Load Electricity demand (consumption), measured in MW or MWh.
    Fuel Mix The breakdown of electricity generation by fuel source (gas, coal, nuclear, wind, solar, etc.).
    HDD / CDD Heating Degree Days / Cooling Degree Days -- measures of how much heating or cooling is needed relative to a base temperature (typically 65 deg F). Higher HDD = colder weather = more heating demand.
    WDD Weighted Degree Days -- degree days weighted by population or load to better reflect energy demand impact.
    Outage A period when a power plant is not generating, either planned (maintenance) or forced (breakdown). Measured in MW offline.
    Tie Flow Electricity flowing between grid regions via interconnection points (ties).
    Hub A trading point where electricity prices are set (e.g., Western Hub, Eastern Hub in PJM).
    Pricing Node A specific location on the grid where LMPs are calculated.
    RTO Regional Transmission Organization -- often used interchangeably with ISO. PJM is technically an RTO.
    Respondent In EIA data, the entity reporting generation data (e.g., PJM, ERCO for ERCOT, CISO for CAISO).
    BCF Billion Cubic Feet -- a unit for measuring natural gas volumes.
    SFTP Secure File Transfer Protocol -- used to exchange files with counterparties (brokers, prime brokers).
    NAV Net Asset Value -- here refers to the NAV fund administrator who provides position reports.
    Marex A futures broker that provides position and trade data.
    Clear Street A prime broker that provides trade confirmation data.

    Data Terms¶

    Term Definition
    Scrape An automated script that pulls data from an external source (API, SFTP) and loads it into our database.
    Upsert Insert new rows or update existing rows if they already exist (based on primary key). Prevents duplicates.
    dbt "data build tool" -- transforms raw data into clean, analysis-ready views using SQL.
    Mart A dbt model at the final layer, ready for consumption by dashboards and analysts.
    Staging A dbt model that cleans and renames raw source data but doesn't join tables.
    Source In dbt, a reference to a raw table loaded by a scrape.
    Ephemeral A dbt model that doesn't create a table/view; it's inlined as a CTE into downstream models.
    View A virtual table defined by a SQL query. Refreshes every time you query it.
    Primary Key The column(s) that uniquely identify each row. Used by upsert to detect duplicates.
    Grain What one row represents (e.g., one row per hour per region).
    Pipeline Run Logger Internal tracking system that records when each scrape ran, whether it succeeded, and how many rows were processed.
    Prefect The workflow orchestration tool that schedules and monitors scrape pipelines.
    T+1 Data available one business day after the event (e.g., end-of-day positions available next morning).
    Model Run In weather/forecast context, a specific execution of a forecast model (e.g., GFS 00Z run).
    GFS Global Forecast System -- a U.S. government weather model.
    ECMWF European Centre for Medium-Range Weather Forecasts -- a leading global weather model.
    AIFS Artificial Intelligence/Integrated Forecasting System -- ECMWF's AI-based weather model.
    Ensemble (ENS) A set of slightly varied forecast runs used to gauge forecast uncertainty.
    Operational (OP) The primary, highest-resolution run of a weather model.
